# Universal-Translation-Search-Bookmarklet
Bookmarklet for Yapla translation pages that loads all paginated entries and enables fast client-side search across features, keys, and translation values. Designed to quickly locate strings in large translation tables without navigating page by page.

##Description

A lightweight browser bookmarklet that enables full-text search across all pages of a paginated translation table.

This tool automatically loads every page of the translation list, aggregates the results locally, and provides an instant search interface across features, keys, and translation values. It is designed to simplify debugging, auditing, and locating translation strings in large datasets that are normally split across many pages.

The interface loads directly inside the current page and includes controls to load pages, stop the process, and perform fast client-side searches.

Features

Loads all paginated translation entries automatically

Concurrent page loading for faster indexing

Instant client-side search across all columns

Works regardless of language column names (FR, EN, IT, etc.)

Bookmarklet-based: no installation required

Lightweight floating UI panel

Error-tolerant page loading

Built for large translation datasets

Usage

Save the script as a browser bookmarklet.

Open the translation management page.

Click the bookmark.

Load all pages and search instantly across the dataset.
